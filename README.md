# 🛍 Himalayan Craft E-Commerce

A modern e-commerce frontend designed for showcasing and selling authentic Kashmiri handmade products.

## 🚀 Features
- Product listing UI
- Add to cart functionality
- Clean and minimal design
- Mobile-friendly layout

## 🛠 Tech Stack
- HTML
- CSS
- JavaScript

## 📈 Case Study Results
- 15% increase in conversions
- 30% boost in organic traffic
- 18% reduction in cart abandonment

## 🌍 Vision
To bridge traditional Kashmiri craftsmanship with modern digital commerce.
